# docs-update-rules

> 适用于全文档替换、命令同步、清理残留、统一术语。

## 触发关键词

出现以下需求时，按全量文档任务处理：

- `所有文档`
- `全部更新`
- `全量同步`
- `清理残留`
- `统一改掉`
- `全文替换`

## 执行流程

1. 先扫描全部相关文档。
2. 列出命中文件清单。
3. 再逐个修改。
4. 修改后再次扫描确认无残留。
5. 回复里给出：
   - 命中文件
   - 已修改文件
   - 是否确认无残留
   - 若有残留，残留位置在哪

## 强制要求

- 未复扫前，不得说“所有文档已更新”
- 未列清单前，不得说“已经全部处理”
- 不允许只改几个明显文件就结束

## 推荐表述

- `我先全量扫描相关文档`
- `命中文件如下...`
- `已完成修改，正在复扫残留`
- `复扫后仍有以下残留...`
