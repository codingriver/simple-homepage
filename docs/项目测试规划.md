## 3. 当前执行口径

### 3.1 当前默认执行命令

```bash
npx playwright test --project=chromium
```

### 3.2 当前默认结论适用范围

- 适用于桌面端 Web 页面级 full E2E 覆盖判断
- 不等于移动端已执行覆盖
- 不等于 middleware / shared 已覆盖
- 不等于所有 CLI 分支都已穷尽覆盖

```bash
npx playwright test --project=mobile-chrome
```
